# 组件系统表单定义

## 组件基类表单

### 创建组件

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `name` | `std::string` | 组件名称 | 是 | - |
| `node` | `Node*` | 所属节点 | 是 | - |
| `uuid` | `std::string` | 组件唯一标识符 | 否 | "" |

### 组件状态管理

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `enabled` | `bool` | 是否启用 | 是 | true |

### 组件生命周期

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `deltaTime` | `float` | 时间步长 | 是 | 0.0f |

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
