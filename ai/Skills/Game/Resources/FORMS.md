# Game 类表单定义

## 初始化游戏表单

### 窗口平台初始化

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `window` | `Window*` | 窗口实例 | 是 | - |
| `uiDesignWidth` | `int` | UI设计宽度 | 是 | 1920 |
| `uiDesignHeight` | `int` | UI设计高度 | 是 | 1080 |
| `fitMode` | `UIDesignFitMode` | 适配模式 | 是 | `None` |

### Android平台初始化

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `android` | `Android*` | Android实例 | 是 | - |
| `uiDesignWidth` | `int` | UI设计宽度 | 是 | 1280 |
| `uiDesignHeight` | `int` | UI设计高度 | 是 | 720 |
| `fitMode` | `UIDesignFitMode` | 适配模式 | 是 | `None` |

## 场景管理表单

### 打开场景

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `scene` | `Scene*` | 要打开的场景 | 是 | - |

### 销毁场景

无参数

## 视图管理表单

### 调整视图大小

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `width` | `int` | 新的宽度 | 是 | - |
| `height` | `int` | 新的高度 | 是 | - |

## 调度系统表单

### 安排定时任务（重复执行）

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `func` | `Func` | 回调函数 | 是 | - |
| `instance` | `T*` | 实例指针 | 是 | - |
| `interval` | `float` | 执行间隔（秒） | 是 | 0.0 |

### 安排定时任务（执行一次）

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `func` | `Func` | 回调函数 | 是 | - |
| `instance` | `T*` | 实例指针 | 是 | - |
| `interval` | `float` | 延迟执行时间（秒） | 是 | 0.0 |

### 取消定时任务

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `scheduleID` | `int` | 调度任务ID | 是 | - |

## 输入处理表单

### 更新鼠标状态

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `button` | `int` | 鼠标按钮（0:左键 1:右键 2:中键） | 是 | - |
| `action` | `int` | 鼠标操作（0:抬起 1:按下） | 是 | - |
| `mods` | `int` | 键盘修饰键（0:无 1:Shift 2:Ctrl 3:Alt） | 是 | 0 |

### 更新鼠标位置

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `xpos` | `double` | 鼠标X坐标 | 是 | - |
| `ypos` | `double` | 鼠标Y坐标 | 是 | - |

### 更新键盘状态

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `key` | `int` | 键盘键值 | 是 | - |
| `scancode` | `int` | 键盘扫描码 | 是 | - |
| `action` | `int` | 键盘操作（0:抬起 1:按下） | 是 | - |
| `mods` | `int` | 键盘修饰键（0:无 1:Shift 2:Ctrl 3:Alt） | 是 | 0 |
